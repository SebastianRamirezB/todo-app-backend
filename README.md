# REST API

Este es un servidor REST API desarrollada para la aplicación Todo App Challenge. puede ver el demo aquí: [Demo](https://todo-app-se.herokuapp.com/)

## Tecnologias usadas

* MongoDB
* Express
* Node
* JSON Web Tokens

## Instalación

1. Clona este proyecto.
2. Ve a la carpeta del proyecto `cd ???`
3. Instala las dependencias `npm install`
5. Para el funcionamiento del servidor debe tener una cuenta de mongodb y configurar la cadena de conexión en las variables de entorno .env
4. Corre el ambiente local `node index.js`
